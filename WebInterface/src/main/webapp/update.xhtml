Update existing data with a new CSV file

You can upload an updated version of the data. Please consider that you can only update CSV files which have the same structure as the original file had. There are several modes available. In some scenarios, only new data needs to be appended to an existing dataset. In this case new records are inserted and the original data is not changed. In other cases you might need to re-upload a file after you made some corrections. In this case the system needs to detect changed records and update them accordingly.

If your CSV file contains headers, please indicate this in the menu below. Otherwise the headers will be inserted as new records.

Step 1: Select the changed data

Select the table that will be updated. Please note that your CSV file needs to be in the very same format as it was during the first upload. Select the file from your hard drive and submit it.
