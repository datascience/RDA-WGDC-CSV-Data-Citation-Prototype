    TimeStamped

    @Entity
public class Organization
extends java.lang.Object
implements TimeStamped
    Organization entity. An organization has a name and a prefix. The prefix needs to be unique within the database. Each organization can set the default values for the length of the identifiers individually. If no values are specified, the default values are read from the property files.
